About {{ site.title }}

{{ site.title }} is:

  • a SVG 1.1 to PNG, PDF, PS and SVG converter;
  • a command-line interface;
  • a Python 3.6+ library;
  • known to work at least on Linux, OS X, and Windows;
  • based on the Cairo 2D graphics library;
  • tested using the W3C test suite;
  • LGPLv3-licensed free software.

Let's go!

Install

{{ site.title }} is available on PyPI, you can install it with pip: 

$ pip3 install cairosvg

You can have more information and help in the "Installation" part of the documentation .

Convert

You can use {{ site.title }} as a standalone command-line program: 

$ cairosvg image.svg -o image.png

Embed

You can also use {{ site.title }} as a Python 3 library: 

$ python3
>>> import cairosvg
>>> cairosvg.svg2pdf(url='image.svg', write_to='image.pdf')
